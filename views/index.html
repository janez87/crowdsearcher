About crowdsearcher

CrowdSearcher builds human-powered applications ON crowdsourcing engines and social networks.

CrowdSearcher brings together a design methodology, a specification paradigm and a reactive execution control environment for designing, deploying, and monitoring crowd-based applications. It is focused on data-centric applications, which apply structured human processing to large datasets; these applications represent the largest and currently most important class of crowd-based applications, as they include big data preparation and cleaning, fact checking, opinion mining, marketing campaigns, image recognition and commenting, multimedia decoding and tagging, and so on.

The common aspect of these applications is the interaction between the requestor (who poses a task), the system (which organizes the computation), and a potentially wide set of performers (who are in charge of performing crowd tasks); performers are engaged through a variety of systems, including crowdsourcing platforms, such as Amazon Turk or CrowdFlower, and social networks, such as Facebook, Twitter or LinkedIn.

How it works

Create a task following these simple steps.

  • Task

    Select the task type that suites your needs.

  • Object

    Import the objects that will be enriched and/or evaulated.

  • Execution

    Select the platform where you want to execute your task.

  • Invitation

    Select the platform for inviting the performers.

  • Review

    Review and create the task.
